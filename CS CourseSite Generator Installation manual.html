CS CourseSite Generator Installation manual

Base Requirements

In order to use CS CourseSite Generator the following prequisites are necessary.
  • Apache Web Server (You can also use other web servers but it will reduce security of your website).
  • PHP version 4.3.0 Until 7.0.4 or better installed as an apache module.
  • Linux/Unix Server

Installing the package

To create a website for your course do the followings:
  1. Create a directory for your course on the server.
  2. Run this command on linux shell:
        chmod 777 your-course-directory
  3. Download install.php and CoursePackage.tar.gz and copy them to your-course-directory.
  4. In your web browser go to http://course-website/install.php
    Setup will install the package and will set configurations.
  5. After the installation completed run this command on linux shell:
       chmod 755 your-course-directory
